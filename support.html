ğŸ”§ Troubleshooting Common Issues

Problem: Can't find a transaction

  • Check if you're in the right time view (Day/Week/Month)
  • Look for the transaction in a different category
  • Use the trash icon to check deleted items

Problem: Budget percentage seems wrong

  • Make sure you've set your correct monthly budget
  • Check that all transactions are in the right month
  • The percentage resets each month

Problem: Saving plan not updating

  • Make sure to tap "Save" after adding money
  • Check that you entered the amount correctly
  • Progress updates immediately after saving
